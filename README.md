# ProFich

Sistema independiente de registro horario para pequeños negocios, concebido como proyecto de portfolio y para su uso real en un establecimiento de hostelería.

## Objetivo

ProFich permitirá registrar la jornada laboral de forma clara y trazable, contemplando las necesidades habituales de hostelería: entradas, salidas, pausas, turnos partidos e incidencias.

## Alcance previsto

- Fichaje mediante PIN personal.
- Registro de entradas, pausas, reanudaciones y salidas.
- Soporte para turnos partidos.
- Gestión de incidencias y correcciones justificadas.
- Historial de auditoría.
- Resúmenes diarios y mensuales.
- Exportación de registros.
- Acceso diferenciado para empleados, encargados y administradores.

## Estado

Proyecto en fase inicial de definición. Todavía no se ha incorporado el código de la aplicación.

## Tecnologías previstas

- Java
- Spring Boot
- Spring Security
- JPA / Hibernate
- Base de datos relacional
- Docker

La selección definitiva de tecnologías y la arquitectura se documentarán antes de comenzar el desarrollo.
